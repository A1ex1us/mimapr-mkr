# mimapr-mkr

Модели и методы анализа проектных решений

## Описание проекта

Этот проект представляет собой программное решение для анализа проектных решений, разработанное на языке C++. Проект включает в себя основные компоненты для обработки и анализа данных.

## Структура проекта

- `main_fixed.cpp` - основной файл программы
- `header.h` - заголовочные файлы с объявлениями функций
- `operations.cpp` - реализация операций обработки данных
- `edumma_lab_2024_rk6_64b_shliukovap.pdf` - документация/задание проекта

## Требования

- Компилятор C++ (например, g++)
- Система сборки (make или подобная)

## Сборка проекта

Для компиляции проекта используйте команду:

```bash
g++ main_fixed.cpp operations.cpp -o program -std=c++20
```

## Запуск

После успешной компиляции запустите программу:

```bash
./program
```
